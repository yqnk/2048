# 2048-cli

> C++20 2048 CLI game developped with C++ in mind.

## Getting Started

### Prerequisites

- Linux
- [GNU make](https://www.gnu.org/software/make/manual/make.html) (>=4.4.1) (only tested on 4.4.1...)

### Building

    $ git clone git@github.com:yqnk/2048-cli.git
    $ make

### Executing

    $ ./main

### In-game commands
- z : moves up
- q : moves left
- s : moves down
- d : moves right
- x : exits
